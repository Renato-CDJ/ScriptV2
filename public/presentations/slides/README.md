# Slides Convertidos

Para exibir apresentações com navegação slide por slide, converta seu arquivo PowerPoint em imagens individuais e coloque-as aqui.

## Estrutura de Pastas

\`\`\`
public/presentations/slides/
  └── nome-do-arquivo/
      ├── slide-001.png
      ├── slide-002.png
      ├── slide-003.png
      └── ...
\`\`\`

## Como Converter PPT para Imagens

### Opção 1: PowerPoint (Windows/Mac)
1. Abra o arquivo PPT no PowerPoint
2. Arquivo > Exportar > Alterar Tipo de Arquivo
3. Escolha "PNG" ou "JPEG"
4. Salve - cada slide será salvo como imagem separada

### Opção 2: Google Slides
1. Faça upload do PPT para Google Drive
2. Abra com Google Slides
3. Arquivo > Download > PNG (.png) ou JPEG (.jpg)
4. Cada slide será baixado como imagem individual

### Opção 3: LibreOffice (Gratuito)
1. Abra o PPT no LibreOffice Impress
2. Arquivo > Exportar...
3. Escolha "PNG" ou "JPEG" como formato
4. Marque "Exportar todos os slides"

## Nomeação dos Arquivos

Use nomes sequenciais para os slides:
- `slide-001.png`, `slide-002.png`, `slide-003.png`
- Ou: `001.png`, `002.png`, `003.png`

O sistema detectará automaticamente e exibirá em ordem.
